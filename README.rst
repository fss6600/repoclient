==================================
Обновление ЕИИС 'Соцстрах'. Клиент
==================================

Клиент для обновления подсистем `ЕИИС 'Соцстрах'`.

Клиент предназначен для установки, обновления и удаления подсистем на рабочих местах пользователей. Обновление происходит по принципу односторонней синхронизации файлов из репозитория, который расположен на сетевом ресурсе в сети. *Репозиторий* представляет из себя проиндексированную определенным образом папку (директорию) с открытым доступом по сети, в которой находятся папки с подсистемами `ЕИИС 'Соцстрах'` и иными программами в актуальном состоянии (*пакеты*).

Репозиторий предварительно должен быть создан с помощью программы ``eiisrepo``
